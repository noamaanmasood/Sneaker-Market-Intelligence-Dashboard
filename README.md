# 👟 Sneaker Market Intelligence Dashboard

A data-driven analytics dashboard that helps analyze sneaker resale value, profit trends, and investment opportunities using real market data.

🌐 **Live Demo:** https://sneaker-market-intelligence-dashboard-r9cuv82b7rpnyprx5suxrt.streamlit.app

---

## 📊 Overview

This project analyzes sneaker market data to uncover:
- 💰 Resale profit opportunities
- 📈 High-performing sneaker models
- 🏷️ Brand-wise performance comparison
- 🧠 Investment insights for sneaker reselling

It transforms raw sneaker dataset information into an interactive dashboard for better decision-making.

---

## 🚀 Features

- 🔍 Filter sneakers by brand and model
- 💸 Calculate resale profit dynamically
- 📊 Interactive bar charts for profit analysis
- 🏆 Top profitable sneaker rankings
- 🎛️ Sidebar-based filtering system
- 🌙 Clean dark-themed UI
